# 温湿度实时监控spring boot后台代码
### 环境
- 操作系统：Windows10
- 数据库：MySQL 8.0.21
- 数据库开发工具：SQLyog/Navicat
- 集成开发环境：IntelliJ IDEA 2019.3.1
- 前端框架：Vue
- 异步请求工具：axios
- 前端样式UI：element-ui
- 后台框架：Spring Boot
- 持久层框架：mybatis-plus
- 数据验证工具：Annotation JSR -303 标准
### 功能描述
1. 用户增删改
2. 登录权限控制
3. 用户登录注册
4. 温湿度可视化分析
5. 异常数据监控
6. 用户日志查看
7. 分页以及多条件查询
